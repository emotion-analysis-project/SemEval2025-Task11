
# SemEval 2025 Task 11: Bridging the Gap in Text-Based Emotion Detection

This dataset is created for SemEval 2025 Task 11, which focuses on text-based emotion detection in 32 languages. The dataset provides annotated textual instances in train, dev and test.

Visit the task paper here for more detail: https://github.com/emotion-analysis-project/SemEval2025-task11
